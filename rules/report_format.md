# Agent Progress Report Format

Every agent MUST maintain a personal report file named `{agent-name}-report.md` in the root directory. This file must be updated after every significant task. Agents are encouraged to read other agents' reports to stay synced.

## Structure
### 1. Current Status
* [Briefly describe the exact task you are working on right now]

### 2. Done
* [Task 1] - Completed on [Date/Time]
* [Task 2] - Completed on [Date/Time]

### 3. To Do (Next Steps)
* [ ] [Pending Task 1]
* [ ] [Pending Task 2]

### 4. Blockers / Dependencies
* [Note any blockers or things you need from the Frontend/Backend/Orchestrator agents]
